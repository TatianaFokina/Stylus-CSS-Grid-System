What is it

This is Bootstrap 4 grid system based on CSS Grid with Stylus mixins.

How it works

Bootstrap grid system based on a 12 column layout and has multiple tiers, one for each media query range. The grid uses a series of containers, rows, and columns to layout and align content. It’s built with CSS Grid and is fully responsive.

  • Containers provide a means to center and horizontally pad your site’s contents. Use .container for a responsive pixel width or .container-fluid for width: 100% across all viewport and device sizes.
  • The above example creates three equal-width columns on extra small (xs), small (sm), medium (md), large (lg) and extra large (xl) devices using our predefined grid classes.
  • Columns are centered in the page with the parent .container.
  • Rows are horizontal groups of columns that ensure your columns are lined up properly.
  • Content should be placed within columns, and only columns may be immediate children of rows.
  • Columns without a set width will automatically layout with equal widths. For example, four instances of .col-sm will each automatically be 25% wide for small breakpoints.
  • Column classes indicate the number of columns you’d like to use out of the possible 12 per row. So, if you want three equal-width columns, you can use .col-sm-4.
  • Column widths are set in percentages, so they’re always fluid and sized relative to their parent element.
  • Columns have row and column gap to create the gutters between individual columns. You can customize it and change properties of grid-column-gap and grid-row-gap.
  • Grid tiers are based on minimum widths, meaning they apply to that one tier and all those above it (e.g., .col-sm-4 applies to small, medium, large, and extra large devices).

Auto-layout columns

Utilize breakpoint-specific column classes for equal-width columns. Add any number of unit-less classes for each breakpoint you need and every column will be the same width.

Responsive classes

Bootstrap’s grid includes five tiers of predefined classes for building complex responsive layouts. Customize the size of your columns on extra small, small, medium, large, or extra large devices however you see fit.
